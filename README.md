# わが家の家計簿

iPhone で使う、自分専用の家計簿アプリです。複数の銀行・カードの明細を取り込んで、家計の収支から確定申告の概算までを一つにまとめました。すべてのデータは端末の中だけに保存され、外部には一切送信されません。

-----

## できること

### 明細の自動取込

銀行・カードからダウンロードした CSV を選ぶだけで、形式を自動で判別して取り込みます。

- 三井住友銀行・イオン銀行（入出金明細）
- Olive（三井住友カード）・セゾンカード・PayPayカード・イオンカード（利用明細）

文字コードや全角・半角の違いも自動で吸収し、同じ取引は重複を除外します。

### 自動でカテゴリ分類

店名や摘要のキーワードから、食費・外食・医療・薬・旅行などのカテゴリを自動で振り分けます。分類はワンタップで修正でき、「今後もこの分類」を登録すれば次回から自動で学習します。

### カードの未払金を自動計算

各カードの請求額を「カード未払金（負債）」として自動で集計し、カード別の内訳も確認できます。銀行側のカード引落は二重計上にならないよう自動で振替（対象外）に分類されます。

### 収支をいろいろな切り口で

月・年・指定期間（◯月〜◯月）で、収入・支出・差引とカテゴリ別の内訳をドーナツグラフで表示します。複数カテゴリをまとめて絞り込むこともできます。

### 資産（純資産）の管理

口座ごとの月末残高から純資産を自動で計算し、推移をグラフで表示します。銀行残高は CSV 取込時に自動記録、現金や積立金などは手入力で管理できます。

### 確定申告の概算

給与＋年金の所得税を計算します。令和7・8年分の税制改正に対応し、医療費やふるさと納税は家計簿から自動で集計。還付・納付の見込み額や、ふるさと納税の限度額の目安も表示します。

### 医療費集計フォームの出力

税タグを付けた医療費を、国税庁の公式フォーム（Excel）に書き出します。確定申告書等作成コーナーや e-Tax にそのまま取り込めます。

### 書類の保管

年金通知書・源泉徴収票・控除証明書などの PDF や画像を、年度別・種類別に端末内に保管できます。申告のときにすぐ取り出せます。

### 見やすさへの配慮

文字サイズを標準・大・特大の3段階で切り替えられます。アプリ全体に反映されます。

-----

## プライバシー

すべての明細・残高・書類は **この iPhone の中だけ** に保存されます。外部のサーバーには一切送信されません。家計のデータを安心して預けられる、自分専用の置き場所です。

機種変更などに備えて、設定からデータをバックアップ（JSON）として書き出すこともできます。

-----

## 確定申告について

申告タブの計算は概算です。令和7年度の税制改正を反映し、実際の申告書と一致することを確認していますが、端数処理や個別の事情で差が出る場合があります。最終確認は国税庁「確定申告書等作成コーナー」で行ってください。

-----

iPhone のホーム画面に追加すれば、普通のアプリのように起動できます。
